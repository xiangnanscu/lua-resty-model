# lua-resty-model

Openresty super postgresql orm finally comes out! Inspired by django orm. Support arbitrary depth of automatic join, group by, reversed lookup, F, Q, annotate, etc.

## Install

```sh
opm get xiangnanscu/lua-resty-model
```
